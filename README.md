# openslide-to-png
OpenSlide: Uses the [OpenSlide](https://openslide.org/) library to convert whole-slide image files to .png for viewing in Flywheel. 

Supported file types include:
* Aperio (.svs, .tif)
* Hamamatsu (.ndpi, .vms, .vmu)
* Leica (.scn)
* MIRAX (.mrxs)
* Philips (.tiff)
* Sakura (.svslide)
* Trestle (.tif)
* Ventana (.bif, .tif)
* Generic tiled TIFF (.tif)
